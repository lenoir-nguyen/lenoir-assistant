# Lenoir Chatbot Project Prompt

## What to Build

A **multilingual AI chat SPA** that grows through 5 versions. Supports conversations in English, French, and Vietnamese via GPT-4o.

**Tech**: Next.js 14 (frontend) + FastAPI (backend), deployed to Vercel + Railway with auto-deploy on push to main.

## Core Features by Version

| Version | Features | Status |
|---------|----------|--------|
| **v1** | Text chat, language selector (en/fr/vi), client-side history | ✅ Done |
| **v2** | Voice input (Whisper) + voice output (TTS) | ⏳ Planned |
| **v3** | Authentication (passphrase + PIN), owner/stranger modes | ⏳ Planned |
| **v4** | PostgreSQL + LangChain memory persistence | ⏳ Planned |
| **v5** | RAG system (pgvector, embeddings, personal facts) | ⏳ Planned |
